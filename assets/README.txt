# Asset Instructions

Please add your images to this folder with the following names to have them automatically appear on the website:

## Profile/Hero
- `hero-profile.jpg` (Approx 400x400px)

## Flyers
- `flyer-1.jpg` (Event Flyer)
- `flyer-2.jpg` (Business Flyer)
- `flyer-3.jpg` (Party Flyer)

## Logos
- `logo-1.png`
- `logo-2.png`
- `logo-3.png`
- `logo-4.png`

## Posters
- `poster-1.jpg` (Album Art)
- `poster-2.jpg` (Concert Poster)
- `poster-3.jpg` (Movie Poster)

Note: You can use .jpg, .png, or .webp formats. If you use a different format, please update the src attributes in `index.html`.
